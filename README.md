# Mi Taller al Día - Sistema de Gestión

Un sistema integral de control de taller y punto de venta (POS) desarrollado a medida para la Zapatería y Maletería Daniel. 

Este proyecto busca digitalizar el flujo de trabajo de un negocio tradicional de reparación de calzado, reemplazando el registro manual con una solución tecnológica altamente accesible. La interfaz está diseñada con un enfoque de usabilidad para adultos mayores, priorizando la claridad visual, el alto contraste y la simplificación de procesos operativos.

## Características Principales

* Directorio ágil de clientes para asignación rápida.
* Control visual de inventario de insumos del taller.
* Seguimiento de órdenes de trabajo mediante control de estados (Pendiente, En Proceso, Listo).
* Generación automática de notas de venta y gestión de saldos.
* Automatización de respaldos de la base de datos local.

## Stack Tecnológico

* Frontend: React, Vite, Tailwind CSS
* Backend: Node.js, Express
* Base de Datos: PostgreSQL 16
* Flujo de Trabajo: Git Flow
